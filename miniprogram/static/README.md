# 静态资源说明

## 需要的图片资源

### 应用图标
- `logo.png` - 应用Logo（已存在）

### 用户相关
- `default-avatar.png` - 默认头像（100x100px）

### 占位图
- `placeholder-vehicle.png` - 车辆占位图（400x300px）
- `placeholder-route.png` - 路线占位图（400x300px）

### TabBar图标
请查看 `tabbar/README.md` 了解详情

## 图片规范

- 格式：PNG（支持透明背景）或 JPG
- 压缩：使用 TinyPNG 等工具压缩以减小体积
- 命名：使用小写字母和连字符，如 `default-avatar.png`

## 临时方案

在正式图片准备好之前：
1. 可以使用纯色背景作为占位
2. 使用 uni-icons 图标组件
3. 使用在线图片服务（如 placeholder.com）

## 1. 产品概述

### 1.1 产品定位

叨叨房车小程序专注于房车租赁服务，为用户提供便捷的房车预订、营地预订、房车旅游、社区互动等一站式服务。

### 1.2 核心业务模块

| 业务模块 | 功能描述 | 目标用户 |
|----------|----------|----------|
| **房车租赁** | 多样化房车租赁服务 | 个人旅行者、家庭出游 |
| **特惠租车** | 固定路线特价服务 | 价格敏感用户 |
| **营地预订** | 房车旅行住宿服务 | 房车租赁用户 |
| **房车旅游** | 个性化路线服务 | 高端用户 |
| **社区互动** | UGC内容社区 | 房车爱好者 |
| **众筹房车** | 房车众筹和收益分润 | 众筹车主 |
| **用户中心** | 完整用户服务体验 | 所有用户 |

### 1.3 用户角色

| 用户角色 | 特征 | 核心需求 |
|----------|------|----------|
| **普通用户** | 偶尔租车 | 方便租车、价格优惠 |
| **PLUS会员** | 高频租车 | 专属折扣���优先服务 |
| **众筹车主** | 参与众筹 | 稳定回报、资金安全 |

## 2. 核心业务流程

### 2.1 房车租赁流程

**流程步骤**：
1. **设置查询条件 (首页 -> 房车预订模块)**：用户在首页的“房车预订模块”中设置取车城市、门店、租期等条件。
2. **浏览与筛选车辆 (车辆列表页)**：点击查询后，跳转至“车辆列表页”，浏览和筛选符合条件的车辆。
3. **查看车辆详情 (车辆详情页)**：从列表中选择车辆，进入“车辆详情页”查看详细信息。
4. **确认订单信息 (订单确认页)**：填写预订信息，选择保险和附加服务，进入“订单确认页”。
5. **支付订单**：在订单确认页完成支付，生成订单。
6. **取车使用**：根据订单信息到店取车，开始租赁。
7. **异常处理**：在“我的”->“订单详情”中处理续租、事故报告等异常。
8. **还车完成**：到指定门店还车，完成整个租赁流程。

**关键规则**：
- **强制查询条件**：城市、门店、取还车租期三个条件必须全部设置才能查询车辆
- **异地还车规则**：
  - 支持异地还车的城市由后台配置，部分车型不支持异地还车
  - 异地还车需支付额外费用，费用在订单确认页显示
  - 特惠套餐可能包含免费异地还车服务
- **车辆分组展示**：查询结果按门店距离分组，同门店车辆按价格从低到高排序
- **价格计算规则**：基础租金 + 保险费用 + 附加服务费 + 异地还车费（如适用），不同时段价格不同
- **库存预留机制**：选择车辆后预留30分钟，超时自动释放
- **保险选择强制**：必须选择基础险、标准险或全险中的一种才能完成预订
- **订单取消规则**：
  - 取车前24小时以上：全额退款
  - 取车前24小时内：扣除租金10%作为违约金
  - 取车后：按实际使用天数计算费用，不足1天按1天计算

### 2.1.1 异常情况处理流程

**续租申请流程**：
- **触发场景**：用户在租期内需要延长租赁时间
- **申请规则**：必须在当前租期结束前24小时申请
- **处理流程**：
  1. 用户在订单详情页点击"申请续租"
  2. 选择续租天数，系统按当前价格标准计算续租费用
  3. 填写续租原因，提交申请
  4. 后台审核车辆后续预订情况
  5. 审核通过后，用户支付续租费用
  6. 订单租期自动延长，订单状态更新
- **限制条件**：如车辆后续已被预订，则无法续租

**事故处理流程**：
- **触发场景**：租赁期间发生交通事故或车辆损坏
- **处理流程**：
  1. 用户在订单页点击"事故报告"
  2. 填写事故信息（时间、地点、经过）
  3. 上传现场照片和相关证件
  4. 系统立即通知客服和门店
  5. 客服联系用户，指导现场处理
  6. 根据情况安排道路救援或维修
  7. 后台评估责任和费用
  8. 处理保险理赔，用户承担保险免赔额部分
- **重要规则**：事故发生后必须立即报告，严重事故需报警处理

**车辆故障处理流程**：
- **触发场景**：车辆发生非事故性故障（如机械故障、电器故障）
- **处理流程**：
  1. 用户在订单页点击"道路救援"
  2. 选择故障类型：无法启动、轮胎问题、电池问题、其他故障
  3. 系统自动定位用户位置
  4. 安排最近的合作维修点或道路救援服务
  5. 维修人员现场处理或拖车至维修点
  6. 根据故障情况提供备用车辆或延长租期
- **服务保障**：提供24小时道路救援服务，基础救援费用由平台承担

### 2.2 众筹购买流程

**流程步骤**：
1. **浏览项目**：查看众筹项目列表和详情
2. **选择份额**：确定购买份额数量
3. **确认信息**：填写购买人信息，选择抵扣券
4. **签署协议**：阅读并同意众筹协议
5. **完成支付**：支付购买款项
6. **获取份额**：获得众筹份额，开始收益

**关键规则**：
- **购买数量规则**：单次最低购买1份，单次购买上不封顶，众筹期内可多次购买
- **众筹成功判定**：达到目标金额或众筹期结束即视为成功，未达到目标金额全额退款
- **收益分配规则**：众筹成功次月开始计算收益，每月5号发放上月收益
- **抵扣券使用规则**：仅限持有"众筹车主"标签的用户使用，每笔交易限用1张
- **协议签署规则**：必须阅读并同意众筹协议才能完成支付
- **资金安全规则**：众筹资金由第三方支付平台托管，项目成功后划转至项目方

### 2.3 份额交易流程

**流程步骤**：
1. **浏览市场**：查看份额交易市场信息
2. **选择操作**：选择买入或卖出操作
3. **确认交易**：填写交易信息，确认手续费
4. **执行交易**：完成交易撮合和资金划转
5. **更新份额**：更新份额持有情况

**关键规则**：
- **价格形成机制**：买卖双方报价撮合成交，价格实时更新
- **交易手续费**：买卖双向收取0.5%手续费，最低1元起收
- **风险控制机制**：
  - 价格异常波动时自动熔断暂停交易
  - 大额交易（单笔超过1000份）需要人工审核
  - 恶意操纵市场行为的监控和处罚
- **资金到账规则**：卖出资金T+1日到账，买入即时确认份额

## 3. 功能模块需求

### 3.1 首页模块

**核心功能**：
- **顶部公告栏**：单行滚动公告，展示活动通知、系统公告等重要信息
- **轮播广告**：活动推广和品牌展示（移除搜索功能）
- **房车预订**：取车城市、取车门店、异地还车选项、取还车租期、时间条件设置和查询
- **金刚区**：主要功能模块入口（6个功能入口）
- **推荐内容**：社区内容推荐（营地、房车旅游、热门帖子），众筹项目推荐

**关键业务规则**：
- **公告栏规则**：
  - 单行文字横向滚动显示
  - 优先展示紧急公告和活动通知
  - 点击公告可查看详情
  - 支持多条公告轮播切换
- **查询条件验证**：取车城市、取还车租期三个条件必须全部填写完整才能启用查询按钮
- **异地还车选项**：
  - 勾选异地还车后显示还车城市和门店选择字段
  - 异地还车费用在查询结果页面和订单确认页显示
  - 部分车型不支持异地还车，系统会自动过滤
- **城市定位规则**：首次使用自动获取用户当前城市，支持手动切换城市
- **门店展示规则**：只显示选中城市的门店，按距离用户当前位置由近及远排序
- **推荐内容规则**：
  - 社区内容推荐（营地、房车旅游、热门帖子），营地推荐：显示距离用户100公里内的热门营地
  - **众筹项目推荐**：推荐优质的众筹房车项目
  - 路线推荐：推荐用户所在城市出发的热门房车路线
- **广告位轮播规则**：优先展示当前城市的活动，无活动时展示品牌广告
- **功能入口权限**：未登录用户可浏览所有入口，点击后触发登录引导

### 3.2 房车租赁模块

**核心功能**：
- **房车列表**：按门店分组展示可用车辆
- **房车详情**：车辆信息、配置、评价展示
- **保险选择**：基础险、标准险、全险选择
- **附加服务**：门店特色服务和全局附加服务
- **订单确认**：完整订单信息确认和支付

**关键业务规则**：
- **同城门店规则**：只展示用户选择城市的门店，按门店距离分组，同一门店内车辆按价格排序
- **车辆状态规则**：只显示可租赁状态的车辆，已租出、维护中车辆不展示
- **异地还车标识**：
  - 车辆列表明确标识是否支持异地还车
  - 异地还车费用在车辆详情页和订单确认页显示
  - 部分特惠套餐可能包含免费异地还车服务
- **合作车辆标识**：明确区分自营车辆和合作车辆，标注服务差异
- **保险方案规则**：
  - 基础险：包含第三者责任险，不包含车辆损失险
  - 标准险：包含基础险+车辆损失险（80%赔付）
  - 全险：包含标准险+车辆损失险（100%赔付）+驾意险
- **附加服务分类**：
  - 门店特色服务：特定门店提供的独特服务（如免费接送）
  - 全局附加服务：所有门店都提供的服务（如儿童座椅）
- **价格显示规则**：显示日租金，总价根据租赁天数和异地还车费自动计算
- **车辆操作规则**：
  - 提供【收藏】功能，用户可收藏喜欢的车辆
  - 提供【详情】功能，查看车辆详细信息

## 🎯 核心规范概览

### 胶水编程优先

- 能复用不造轮子，能连接不重写
- 优先使用成熟的开源库和官方 SDK

### 质量红线

- ✅ 必须通过 Linter（零警告零错误）
- ✅ 必须通过所有测试

## 🔧 小程序端特定规范

### 技术栈

- uni-app 3
- Vue 3 Composition API
- TypeScript
- uni-ui 组件库

### uni-app 开发规范

- 遵循 uni-app 官方最佳实践
- 使用条件编译处理平台差异
- 优先使用 uni-ui 组件库

### 开发方式

使用 HBuilderX 开发：

1. 使用 HBuilderX 打开 miniprogram 目录
2. 点击"运行" → "运行到微信开发者工具"

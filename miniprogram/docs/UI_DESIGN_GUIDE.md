# 叨叨房车小程序 UI 设计规范 (UI Design Guide)

**版本**: 1.0.0  
**更新日期**: 2025-12-06  
**设计主题**: Modern Journey (现代旅程)

本以此文档为基准，对"叨叨房车"小程序进行全站 UI 现代化升级。设计核心理念为：**沉浸 (Immersion)、呼吸感 (Airy)、层级 (Hierarchy)**。

---

## 1. 核心视觉原子 (Design Tokens)

### 1.1 色彩系统 (Color System)

保持品牌主色调不变，优化辅助色和中性色，增加通透感。

*   **主色 (Primary)**: `#FF9F29` (温暖橙) - 用于主按钮、关键图标、高亮文本。
*   **渐变主色 (Primary Gradient)**: `linear-gradient(135deg, #FF9F29 0%, #FFB84D 100%)`
*   **次要色 (Secondary)**: `#1A1A1A` (深黑) - 用于主要标题。
*   **背景色 (Background)**: 
    *   `#F8F9FC` (全局背景 - 冷灰白，比纯白更有质感)
    *   `#FFFFFF` (卡片背景)
*   **文本色 (Typography)**:
    *   Primary: `#1D2129` (主要信息)
    *   Secondary: `#4E5969` (次要信息)
    *   Placeholder: `#86909C` (辅助/占位)
*   **功能色**:
    *   Error: `#FF4D4F`
    *   Success: `#00B578`
    *   Warning: `#FF7D00`

### 1.2 阴影系统 (Elevation & Shadows)

摒弃生硬的黑阴影，使用带色相的弥散阴影 (Diffuse Shadows)。

*   **Level 1 (卡片/按钮)**: `0 4rpx 16rpx rgba(0, 0, 0, 0.04)`
*   **Level 2 (悬浮/弹窗)**: `0 8rpx 24rpx rgba(0, 0, 0, 0.08)`
*   **Level 3 (高光/强调)**: `0 12rpx 32rpx rgba(255, 159, 41, 0.15)` (主色阴影)

### 1.3 形状与圆角 (Shapes)

全面采用大圆角，营造亲和力。

*   **卡片 (Card)**: `24rpx`
*   **按钮 (Button)**: `50rpx` (全圆角)
*   **小组件 (Widget)**: `12rpx` 或 `16rpx`

### 1.4 间距 (Spacing)

遵循 8px (16rpx) 栅格系统。

*   **页面边距**: `32rpx`
*   **模块间距**: `24rpx`
*   **内容间距**: `16rpx`

## 2. 组件设计规范 (Component Specs)

### 2.1 导航栏 (Navbar)

*   **风格**: 沉浸式。
*   **行为**: 初始背景透明，内容（白色/黑色）根据背景图适配；滚动后背景变为 `rgba(255,255,255, 0.95)` + 高斯模糊 (Backdrop Blur)。

### 2.2 卡片 (Cards)

*   **通用样式**: 白色背景，无边框（或极淡的 `1px solid #F2F3F5`），Level 1 阴影。
*   **交互**: 点击时缩放 `scale(0.98)`。

### 2.3 按钮 (Buttons)

*   **主按钮**: 品牌色渐变背景，白色文字，Level 3 阴影。宽度通常为 100% 或固定宽度。
*   **次级按钮**: 灰色背景 (`#F2F3F5`)，深色文字。
*   **文字按钮**: 无背景，主色文字。

### 2.4 列表项 (List Items)

*   **布局**: 图片左置 (16:9 或 4:3)，右侧信息垂直排列。
*   **分隔**: 移除全宽分割线，使用内缩进分割线或直接利用间距分隔。

## 3. 页面特定规范 (Page Specifics)

### 3.1 首页 (Home)

*   **Hero区域**: Banner 图铺满顶部（含状态栏），高度约 `500rpx`。
*   **预订表单**: 悬浮于 Banner 之上（负 margin-top），白色背景，强阴影。
    *   输入框去背景化，仅保留文字和图标，中间加分割线。
    *   日期显示为 "MM月DD日" (大字号 DIN 字体) + "周X" (小字号)。

### 3.2 车辆列表 (Vehicle List)

*   **布局**: 单列大卡片。
*   **图片**: 宽幅图片，展示车辆侧颜。
*   **标签**: 胶囊形标签，淡色背景+深色文字。

### 3.3 个人中心 (Profile)

*   **头部**: 杂志风格排版，取消明显的色块边界，头像与背景融合。
*   **菜单**: 宫格图标 (Grid) 替代长列表 (List)，提高空间利用率。

### 3.4 订单中心 (Order)

*   **状态标识**: 卡片左侧设置 6rpx 宽的彩色状态条 (Status Bar)。
*   **布局**: 关键信息（时间、车型）对齐，次要信息灰度处理。

### 3.5 社区 (Community)

*   **流式布局**: 瀑布流 (Masonry) 展示图片。
*   **操作栏**: 点赞/评论图标外置于图片下方或覆盖于图片右下角。

---

## 4. 字体规范 (Typography)

*   **数字/价格**: `DIN Alternate` 或 `Roboto` (加粗)。
*   **大标题**: `36rpx` / `600` weight。
*   **正文**: `28rpx` / `400` weight。
*   **说明文**: `24rpx` / `400` weight / Color: `#999`。

## 1. 产品概述

### 1.1 产品定位

房车租赁服务平台，提供房车预订、营地预订、房车旅游、社区互动、车辆托管等服务。

### 1.2 核心业务模块

| 业务模块 | 功能描述 |
|----------|----------|
| 房车租赁 | 房车预订和租赁服务 |
| 特惠租车 | 固定路线特价套餐 |
| 营地预订 | 营地住宿预订 |
| 房车旅游 | 旅游路线预订 |
| 社区互动 | UGC内容社区 |
| 托管中心 | 车辆托管运营 |
| 会员服务 | PLUS会员特权 |

### 1.3 用户角色

**C端用户**：普通用户、PLUS会员、托管车主
**B端用户**：平台员工、清洁工、服务商、合作方（通过移动管理端注册审核后可登录查看相关数据）

## 3. 核心功能需求

### 3.1 首页模块功能

**核心功能需求**:

- 公告栏：重要信息滚动展示
- 房车预订：城市、门店、租期条件设置
- 功能入口：各业务模块快捷访问
- 内容推荐：社区内容和众筹项目推荐

**关键业务要求**:

- 查询条件必须完整才能启用查询按钮
- 首次使用自动定位用户当前城市
- 营销内容按优先级展示（活动 > 会员 > 优惠券）

### 3.2 房车租赁模块

**核心功能需求**:

- **房车列表**: 按门店分组展示可用车辆
- **房车详情**: 车辆信息、配置、评价展示
- **保险选择**: 基础险、标准险、全险三种方案
- **订单确认**: 完整预订信息确认和支付

**关键业务要求**:

- **车辆展示**: 按门店距离分组，同门店按价格排序
- **保险方案**:
  - 基础险：第三者责任险
  - 标准险：基础险+车辆损失险（80%赔付）
  - 全险：标准险+车辆损失险（100%赔付）+驾意险
- **价格构成**: 基础租金 + 保险费 + 附加服务费 + 异地还车费

**异地还车功能**:

- **功能入口**: 首页预订表单"异地还车"开关
- **功能说明**: 允许用户在不同于取车门店的门店还车
- **UI交互**:
  - 默认状态：异地还车开关关闭，仅显示取车城市/门店选择
  - 开启后：展开还车城市/门店选择区域
  - 取车和还车门店可独立选择
- **费用计算**:
  - 计算公式：`异地还车费 = 两店直线距离(公里) × 单价(元/公里)`
  - 单价标准：2.5元/公里（管理端可配置）
  - 计算示例：北京店 → 深圳店，直线距离约2000公里，费用 = 2.5 × 2000 = 5000元
- **费用展示**:
  - 订单确认页价格明细中单独列出"异地还车费"
  - 显示格式：`异地还车费(北京店→深圳店 约2000km) ¥5000`
  - 总价计算包含异地还车费
- **业务规则**:
  - 同城不同门店也收取异地还车费（按实际距离计算）
  - 特惠租车套餐不支持异地还车（取还车门店固定）
  - 距离计算基于门店GPS坐标的直线距离

#### 3.2.1 订单异常处理功能

**核心功能需求**:

- **续租申请**: 租期内延长租赁时间
- **车辆故障申报**: 车辆故障报修和道路救援
- **交通事故申报**: 交通事故报案和保险理赔

**1. 续租申请功能** (pages/order/renewal.vue)

**功能入口**:
- 订单详情页"续租申请"按钮
- 仅在订单状态为"使用中"时显示

**核心流程**:
1. 选择延长天数(1-30天可选)
2. 系统校验车辆后续档期是否可延长
3. 实时计算新的还车时间和续租费用
4. 展示费用明细(日租金×天数)
5. 提交续租申请或发起续租支付
6. 更新订单时间轴和还车时间

**业务规则**:
- 续租费用 = 原订单日租金 × 延长天数
- 续租期间保险自动延续,无需额外购买
- 车辆后续有预订时,最多可延长至下一订单开始前1天
- 续租申请需在原还车时间前24小时提交
- 续租费用支付成功后立即生效

**2. 车辆故障申报功能** (pages/order/report.vue)

**功能入口**:
- 订单详情页"故障申报"按钮
- 订单状态为"使用中"或"待还车"时可用

**核心流程**:
1. 选择故障类型(发动机/电路/轮胎/其他)
2. 填写故障描述(支持文字+语音输入)
3. 上传故障照片(至少3张,最多9张)
4. 系统自动获取GPS定位信息
5. 确认联系电话(默认用户手机号)
6. 提交申报,生成故障记录
7. 订单自动进入"挂起"状态

**处理机制**:
- 订单挂起期间租金暂停计算
- 客服接收申报后联系道路救援
- 现场评估故障严重程度
- 计算费用(停运损失+维修费+折旧损失)
- 责任划分(用户付费/平台减免/混合方式)
- 提供处理方案:
  - 方案1: 更换车辆(订单继续)
  - 方案2: 终止订单(退款补偿)
  - 方案3: 其它补偿(订单继续)
- 订单状态恢复或完成

**业务规则**:
- 挂起期间费用按当日车辆价格计算
- 挂起期间费用不参与推广或分润
- 故障车辆维修后进入整备流程
- 用户可查看故障处理进度和结果

**3. 交通事故申报功能** (pages/order/report.vue)

**功能入口**:
- 订单详情页"事故申报"按钮
- 订单状态为"使用中"或"待还车"时可用

**核心流程**:
1. 选择事故类型(单方/双方/多方事故)
2. 填写事故描述(支持文字+语音输入)
3. 上传事故照片(不少于10张,包括全景+特写)
4. 系统自动获取GPS位置信息
5. 确认联系电话(默认用户手机号)
6. 提交申报,生成事故记录
7. 订单自动进入"挂起"状态

**处理机制**:
- 订单挂起期间租金暂停计算
- 客服24小时内报保险公司
- 保险公司进行定损评估
- 安排事故车辆维修
- 计算费用(停运损失+维修费+折旧损失)
- 责任划分(用户付费/平台减免/保险理赔/混合方式)
- 跟踪保险理赔进度
- 提供处理方案:
  - 方案1: 更换车辆(订单继续)
  - 方案2: 终止订单(退款补偿)
  - 方案3: 其它补偿(订单继续)
- 订单状态恢复或完成

**业务规则**:
- 挂起期间费用按当日车辆价格计算
- 挂起期间费用不参与推广或分润
- 事故车辆维修完成后进入整备流程
- 用户可查看事故处理进度和理赔结果
- 保险理赔周期通常为15-30个工作日

**订单状态变化**:
- 正常状态 → 挂起(异常处理期) → 已解决(订单继续) 或 已完成(订单结束)

**道路救援支持**:
- 24小时救援热线: 400-XXX-XXXX
- 支持一键拨打救援电话
- 救援范围: 全国主要城市道路（高速公路不提供道路救援）
- 救援服务: 拖车、换胎、搭电、送油、现场维修

### 3.3 特惠租车模块

**核心功能需求**:

- **套餐列表**: 特价套餐展示和筛选
- **套餐详情**: 取还车门店、价格、租期、时间段详情
- **预订流程**: 取车日期选择、保险服务选择、订单确认
- **自动计算**: 根据固定租期自动计算还车时间

**关键业务要求**:

- **全要素固定**: 取车门店、还车门店、价格、租期完全固定

## 4. 业务流程

### 4.1 房车租赁流程

**流程步骤**：

1. **设置查询条件**（首页）: 城市、门店、租期
2. **浏览筛选车辆**（车辆列表页）: 按门店分组展示
3. **查看车辆详情**（车辆详情页）: 配置、价格、评价
4. **确认订单信息**（订单确认页）: 保险选择、附加服务
5. **完成支付订单**: 支付选择、订单生成
6. **取车使用**: 到店取车、开始租赁
7. **异常处理**: 续租申请、事故/故障申报
8. **还车完成**: 门店还车、结束租赁

**异常处理子流程**：

#### 续租申请

租期内用户从订单详情进入续租入口，选择额外天数→系统校验车辆是否可延长→展示新的还车时间与费用→发起续租支付或提交人工审核 → 更新订单时间轴。

#### 车辆故障申报（订单进行中）

**申报流程**：

1. **故障申报入口**：订单详情页点击"故障申报"按钮
2. **选择故障类型**：发动机故障、电路故障、轮胎故障、其他故障
3. **填写故障描述**：详细描述故障情况（支持文字+语音输入）
4. **上传故障照片**：拍摄故障部位照片（至少3张）
5. **自动定位**：系统自动获取GPS位置信息
6. **联系方式确认**：确认联系电话，方便客服联系
7. **提交申报**：提交后生成故障记录，通知客服和门店

**处理流程**：

- 订单自动进入"挂起"状态，记录挂起时间点
- 挂起期间租金暂停计算（不计入订单费用）
- 客服接收申报后，联系道路救援服务
- 现场评估故障严重程度
- 安排故障车辆维修
- 计算费用（停运损失、维修费、车辆折旧损失等）
- 责任划分（用户付费、平台减免、混合方式）
- 提供处理结果供用户选择：
  - **方案1：更换车辆**（订单继续）- 平台安排替换车辆，订单正常进行
  - **方案2：终止订单**（退款补偿）- 订单完成，退还剩余租金并补偿
  - **方案3：其它补偿**（订单继续）- 延长租期、费用减免等，订单继续
- 订单状态恢复（根据处理结果，订单继续或完成）
- 故障车辆维修后进入整备流程

**订单状态变化**：

- 进行中 → 挂起（异常处理期）→ 已解决（订单继续）或 已完成（订单结束）

**费用说明**：

- 挂起期间费用按当日车辆价格计算（停运损失+维修费+折旧损失）
- 挂起期间费用为单独费用，不参与推广或分润计算
- 费用承担方根据责任划分确定

#### 交通事故申报（订单进行中）

**申报流程**：

1. **事故申报入口**：订单详情页点击"事故申报"按钮
2. **选择事故类型**：单方事故、双方事故、多方事故
3. **填写事故描述**：详细描述事故经过（支持文字+语音输入）
4. **上传事故照片**：拍摄事故现场照片（不少于10张，包括全景+特写）
5. **自动定位**：系统自动获取GPS位置信息
6. **联系方式确认**：确认联系电话，方便客服和保险公司联系
7. **提交申报**：提交后生成事故记录，通知客服、门店和保险公司

**处理流程**：

- 订单自动进入"挂起"状态，记录挂起时间点
- 挂起期间租金暂停计算（不计入订单费用）
- 客服接收申报后，24小时内报保险公司
- 保险公司进行定损评估
- 安排事故车辆维修
- 计算费用（停运损失、维修费、车辆折旧损失等）
- 责任划分（用户付费、平台减免、保险理赔、混合方式）
- 跟踪保险理赔进度
- 提供处理结果供用户选择：
  - **方案1：更换车辆**（订单继续）- 平台安排替换车辆，订单正常进行
  - **方案2：终止订单**（退款补偿）- 订单完成，退还剩余租金并补偿
  - **方案3：其它补偿**（订单继续）- 延长租期、费用减免等，订单继续
- 订单状态恢复（根据处理结果，订单继续或完成）
- 事故车辆维修完成后进入整备流程

**订单状态变化**：

- 进行中 → 挂起（异常处理期）→ 已解决（订单继续）或 已完成（订单结束）

**费用说明**：

- 挂起期间费用按当日车辆价格计算（停运损失+维修费+折旧损失）
- 挂起期间费用为单独费用，不参与推广或分润计算
- 费用承担方根据责任划分确定

**关键流程要求**：

- **强制查询条件**: 城市、门店、租期三个条件必须完整
- **异地还车**: 支持部分城市，需额外费用
- **保险选择**: 必须选择基础险、标准险或全险之一
- **库存预留**: 选择车辆后预留30分钟
- **取消政策**:
  - 取车前24小时以上：全额退款
  - 取车前24小时内：扣除10%违约金
  - 取车后：按实际使用天数计算


### 4.2 自有车托管流程

**流程步骤**：

1. **了解托管**: 查看托管中心主页，了解托管政策和收益
2. **提交申请**: 填写车辆信息，上传行驶证和车辆照片（至少12张）
3. **选择门店**: 选择交车门店
4. **填写车主信息**: 填写联系方式和身份信息
5. **提交审核**: 提交托管申请，等待平台审核
6. **审核通过**: 平台审核车辆资质和照片
7. **签署协议**: 线下签署托管协议
8. **交付车辆**: 到门店交付车辆，完成验车
9. **开始托管**: 车辆进入托管运营，开始产生收益

**关键流程要求**：

- **车辆要求**: 车龄≤8年，无重大事故，手续齐全
- **照片要求**: 至少12张实拍照片（外观、内饰、细节）
- **审核时效**: 1-3个工作日完成审核
- **收益分成**: 车主70%，平台30%
- **淡季补贴**: 11.15-3.15期间最高1000元/月保底收益

### 4.3 购车托管流程

**流程步骤**：

1. **浏览车型**: 查看热门托管车型和收益预估
2. **选择车型**: 选择心仪的车型和配置
3. **填写申请**: 填写个人信息和联系方式
4. **提交审核**: 提交购车托管申请
5. **资质审核**: 平台审核申请人资质（年龄、信用、收入）
6. **签署协议**: 审核通过后签署托管协议
7. **购车交付**: 平台协助购车，车辆交付门店
8. **开始托管**: 车辆进入托管运营，开始产生收益

**关键流程要求**：

- **申请人要求**: 年满22周岁，信用良好，有稳定收入
- **保底收益**: 3500元/月保底收益
- **超额分成**: 超过保底部分车主70%，平台30%
- **审核时效**: 3-5个工作日完成资质审核

### 4.4 特惠租车流程

**流程步骤**：

1. **浏览套餐**: 查看特价套餐列表

4. **选择保险服务**: 选择保险方案和附加服务

6. **完成支付**: 支付套餐费用

**关键流程要求**：

- **全要素固定**：取车门店、还车门店、价格、租期完全固定，不可修改

### 4.5 营地预订流程

**流程步骤**：

  - 入住前3天以上：全额退款

### 4.6 房车旅游流程

### 4.7 用户注册登录流程

**流程步骤**：

1. **触发登录**: 业务操作时触发登录引导
2. **选择方式**: 微信授权或手机号登录
3. **微信授权**: 获取微信用户基本信息（头像、昵称）
4. **手机绑定**: 绑定手机号，发送验证码
5. **设置密码**: 设置登录密码（可选）
6. **完善资料**: 补充必要的个人信息

8. **登录成功**: 自动跳转到原操作页面

**多角色登录支持**（新增）：

- **C端用户**（普通用户/PLUS会员/托管车主）：正常使用所有小程序功能
- **B端用户**（平台员工/清洁工/服务商/合作方）：
  - 必须先在管理端或移动管理端注册并通过PC端审核
  - 登录小程序后与C端用户权限完全相同，可正常使用所有功能
  - 允许B端用户登录小程序的目的是方便其通过分享推广等行为获取收益
  - 所有B端业务相关功能（订单处理、任务管理等）在管理端或移动管理端执行

**关键流程要求**：

- **授权策略**: 最小化授权，仅获取必要信息
- **手机验证**: 必须绑定手机号才能进行业务操作
- **密码设置**: 可选择设置密码，支持密码登录
- **资料完善**: 预订等业务需要补充驾驶证等资料
- **登录保持**: 支持记住登录状态，30天内免登录
- **角色权限**: 根据用户角色动态显示功能和数据

### 4.8 车主自用申请流程

4. **选择还车门店**: 可选择同城或异地门店
5. **选择取还车时间**: 选择取车和还车的日期时间

### 4.9 优惠券兑换/购买流程

#### 4.9.1 新人免费领取流程

**流程步骤**：

1. **注册成功**: 用户完成注册后，系统自动发放新人券包
2. **进入特惠商城**: 从首页入口或"我的"页面进入特惠商城
3. **查看新人专区**: 点击"新人专区"查看专属优惠券
4. **选择优惠券**: 浏览新人专享优惠券列表
5. **查看详情**: 点击优惠券查看详细使用规则和适用范围
6. **免费领取**: 点击"免费领取"按钮
7. **领取成功**: 优惠券自动进入"我的优惠券"
8. **使用提醒**: 系统提示优惠券有效期和使用方式

**关键流程要求**：

- **自动发放**: 注册成功后自动发放基础新人券包
- **手动领取**: 新人专区额外优惠券需手动领取
- **领取限制**: 每个新人券仅限领取1次
- **有效期**: 新人券通常有效期较短（7-15天）

#### 4.9.2 积分兑换流程

**流程步骤**：

1. **进入特惠商城**: 从首页入口或"我的"页面进入
2. **查看积分余额**: 顶部卡片显示当前可用积分
3. **浏览优惠券**: 查看各类优惠券，筛选合适的券种
4. **查看详情**: 点击优惠券查看详细使用规则
5. **确认兑换**: 检查积分余额，点击"积分兑换"按钮
6. **二次确认**: 弹窗确认兑换信息（所需积分、优惠券内容）
7. **扣除积分**: 确认后扣除相应积分
8. **兑换成功**: 优惠券自动进入"我的优惠券"
9. **使用优惠券**: 下单时在支付页面选择使用

**关键流程要求**：

- **积分不足提示**: 积分不足时提示如何获取积分（托管收益、评价、签到等）
- **兑换确认**: 必须二次确认，防止误操作
- **即时到账**: 兑换成功后优惠券立即到账
- **兑换记录**: 积分明细中记录兑换历史
- **限领次数**: 部分优惠券限制每人兑换次数

#### 4.9.3 现金购买流程

**流程步骤**：

1. **选择优惠券**: 选择支持现金购买的优惠券
2. **查看详情**: 查看价格和使用规则
3. **点击购买**: 点击"现金购买"按钮
4. **确认订单**: 查看购买信息（价格、数量、有效期）
5. **选择支付方式**: 微信支付/支付宝/余额支付
6. **完成支付**: 支付成功后优惠券自动到账
7. **支付失败处理**: 支付失败时提供重试或取消选项
8. **使用优惠券**: 下单时在支付页面选择使用

**关键流程要求**：

- **价格透明**: 明确显示购买价格和原价对比
- **支付时限**: 15分钟内完成支付，超时订单自动取消
- **支付安全**: 支持密码、指纹、面部识别验证
- **购买记录**: 订单记录中保存购买历史
- **退款规则**: 未使用的优惠券可在购买后24小时内申请退款


#### 4.9.4 积分+现金组合购买流程

**流程步骤**：

1. **选择优惠券**: 选择支持组合购买的优惠券
2. **查看组合价格**: 显示所需积分和现金金额
3. **点击购买**: 点击"组合购买"按钮
4. **确认订单**: 查看积分扣除和现金支付金额
5. **积分验证**: 系统验证积分余额是否充足
6. **选择支付方式**: 选择现金部分的支付方式
7. **完成支付**: 先扣除积分，再完成现金支付
8. **购买成功**: 优惠券自动到账

**关键流程要求**：

- **积分优先**: 先验证积分是否充足，再进行现金支付
- **原子操作**: 积分扣除和现金支付必须同时成功，否则回滚
- **支付失败**: 支付失败时自动退还已扣除的积分
- **价格优势**: 组合购买通常比纯现金购买更优惠
- **限购规则**: 组合购买可能有每人限购次数

#### 4.9.5 分享得券流程

**流程步骤**：

1. **选择分享**: 在优惠券详情页或我的优惠券页面点击"分享"
2. **生成分享内容**: 系统生成分享海报或小程序卡片
3. **选择分享渠道**: 微信好友、微信群、朋友圈
4. **好友接收**: 好友收到分享链接或海报
5. **好友领取**: 好友点击链接进入小程序领取优惠券
6. **领取成功**: 好友成功领取优惠券
7. **奖励发放**: 系统自动向分享者发放奖励券
8. **奖励通知**: 分享者收到奖励到账通知

**关键流程要求**：

- **分享追踪**: 系统记录分享关系，确保奖励准确发放
- **防刷机制**: 同一用户多次领取不重复计算奖励
- **奖励延迟**: 好友领取后即时发放奖励，无需等待使用
- **分享限制**: 每张优惠券可分享次数有限（管理端设定）
- **分享记录**: 查看分享历史和奖励记录

#### 4.9.6 邀请得券流程

**流程步骤**：

1. **进入邀请页面**: 从"我的"页面或特惠商城进入推广中心
2. **生成邀请码**: 系统生成专属邀请码或邀请海报
3. **分享邀请**: 通过微信、朋友圈等渠道分享邀请内容
4. **新用户注册**: 新用户通过邀请链接进入小程序并注册
5. **新用户首单**: 新用户完成首次订单并支付
6. **订单完成**: 新用户订单完成（取车或服务开始）
7. **奖励发放**: 系统自动向邀请人发放专属优惠券
8. **奖励通知**: 邀请人收到奖励到账通知

**关键流程要求**：

- **注册绑定**: 新用户必须通过邀请链接注册才能建立邀请关系
- **首单验证**: 必须完成首单并支付成功才发放奖励
- **防作弊**: 系统检测异常注册和订单，防止刷单行为
- **奖励丰厚**: 邀请奖励通常包含多张不同类型的优惠券
- **邀请记录**: 查看邀请人数、成功注册数、完成首单数、奖励发放记录
- **无上限**: 邀请人数和奖励次数无上限

#### 4.9.7 会员专属领取流程

**流程步骤**：

1. **成为PLUS会员**: 购买PLUS会员资格
2. **进入会员中心**: 从"我的"页面进入会员中心
3. **查看专属券**: 查看会员专属优惠券列表
4. **领取优惠券**: 点击"立即领取"按钮

6. **定期更新**: 每月更新会员专属优惠券

**关键流程要求**：

- **会员验证**: 仅PLUS会员可见和领取
- **定期发放**: 每月1号自动发放当月会员专属券
- **生日特权**: 生日当月额外发放生日专属券
- **会员到期**: 会员到期后已领取的优惠券仍可使用
- **续费奖励**: 会员续费时额外赠送续费专属券

## 5. 业务规则

### 5.1 预订规则

**房车租赁**：

- 强制条件查询（城市、门店、时间）
- 必须选择保险
- 支持取消，按时间段扣除费用
- 取车前24小时以上全额退款
- 取车前24小时内扣除10%手续费
- 取车后按实际使用天数计算

**特惠租车**：

- 固定套餐，不支持个性化修改
- 独立车辆资源，不占用普通租赁
- 特殊取消政策，按时间梯度退款
- 出发前7天以上：全额退款
- 出发前3-7天：退款70%
- 出发前1-3天：退款30%
- 出发当天：不退款

**营地预订**：

- 按天预订，支持多天
- 营位库存实时管理
- 入住前3天以上：全额退款
- 入住前1-3天：退款50%
- 入住当天：不退款

**房车旅游**：

- 按批次组织，需要成团
- 成团前取消全额退款
- 不成团时全额退款或改期
- 出发前7天以上：全额退款
- 出发前3-7天：退款70%
- 出发前1-3天：退款30%

### 5.2 支付规则

**支付方式**：

- 微信支付、支付宝支付、余额支付
- 支持组合支付
- 15分钟内完成支付

**押金规则**：

- 车辆押金：到店支付，还车时退还
- 违章押金：30天后无违章自动退还

**退款规则**：

- 原路退回：支付渠道原路退回
- 退款时效：1-3个工作日到账
- 部分退款：按比例退回各支付渠道

### 5.3 托管规则

**自有车托管规则**：

- 车辆要求：车龄≤8年，无重大事故，手续齐全
- 照片要求：至少2张实拍照片（外观、内饰、细节）
- 审核时效：1-3个工作日完成审核
- 收益分成：车主70%，平台30%
- 淡季补贴：11.15-3.15期间最高1000元/月保底收益
- 托管期限：最短6个月，最长不限

**购车托管规则**：

- 申请人要求：年满22周岁，信用良好，有稳定收入
- 保底收益：3500元/月保底收益
- 超额分成：超过保底部分车主70%，平台30%
- 审核时效：3-5个工作日完成资质审核
- 分期支持：支持12/24/36期分期付款
- 托管期限：与分期期限一致

**车主自用规则**：

- 租金优惠：车主自用租金为0元
- 服务费：需支付门店服务费200-500元/次
- 保险费：已包含在托管协议中，无需额外支付
- 异地还车：支持异地还车
- 审核时效：1个工作日内完成审核
- 取消政策：取车前24小时可免费取消

**收益发放规则**：

- 发放周期：按月发放，次月5号到账
- 结算方式：自动结算到托管车主账户余额
- 提现规则：单次最低提现100元，T+1到账
- 税费处理：平台代扣代缴个人所得税


### 5.4 会员权益规则

**PLUS会员**：

- 会员费：年费99元
- 专属折扣：房车租赁95折
- 免费取消订单特权
- 专属客服支持

**权益叠加规则**：

- 会员折扣 vs 优惠券：会员折扣优先，优惠券在会员折扣后使用
- 多种优惠券：仅限使用1张优惠券，不可叠加使用（日租抵扣券可叠加使用）
- 积分抵扣：积分不可直接使用，只能在特惠商城兑换优惠券
- 特价商品：特惠套餐等特价商品不享受会员折扣

### 5.5 积分系统规则

**积分获得规则**：

- 托管收益：托管车主每月收益的1%转化为消费积分（仅首次托管，包括自有车和购车托管）
- 评价反馈：优质评价获得10-50积分，图片评价额外20积分（限定托管车主）
- 签到打卡：每日签到获得2积分，连续签到额外奖励（限定托管车主）
- 活动参与：参与平台活动获得5-100积分（限定托管车主）
- 推荐好友：成功推荐新用户注册获得50积分（限定托管车主）
- 租车消费：普通用户租车消费100元返1积分

**积分使用规则**：

- 积分兑换：1积分=1元，消费返积分100元返1积分，积分仅可兑换优惠券
- 抵扣限制：不可抵扣现金，仅用于特惠商城兑换优惠券
- 有效期：积分根据用户身份获取，不设定有效期
- 积分转让：不支持积分转让或赠送

### 5.6 内容审核规则

**审核范围**：

- 社区发布内容（攻略、体验、活动、问答）
- 用户评论和评价
- 用户头像和昵称
- 举报内容

**审核标准**：

- 内容相关性：必须与房车、旅游、露营相关主题
- 合法性：符合国家法律法规，禁止违法内容
- 质量要求：内容完整、有价值，避免无意义灌水
- 图片规范：图片清晰、真实，无水印、无侵权

**审核流程**：

- 自动审核：AI系统初步筛查违规内容
- 人工审核：24小时内完成人工审核
- 审核状态：待审核、审核通过、审核拒绝
- 申诉机制：拒绝后可申诉，48小时内重新审核

## 1. 产品概述

### 1.1 产品定位

叨叨房车小程序专注于房车租赁服务，为用户提供便捷的房车预订、营地预订、房车旅游、社区互动等一站式服务。

### 1.2 核心业务模块

| 业务模块 | 功能描述 | 目标用户 |
|----------|----------|----------|
| **房车租赁** | 多样化房车租赁服务 | 个人旅行者、家庭出游 |
| **特惠租车** | 固定路线特价套餐服务 | 价格敏感用户 |
| **营地预订** | 房车旅行住宿服务 | 房车租赁用户 |
| **房车旅游** | 个性化路线服务 | 高端用户 |
| **社区互动** | UGC内容社区 | 房车爱好者 |
| **托管中心** | 闲置房车托管运营服务 | 车主、潜在购车用户 |
| **会员服务** | PLUS会员特权服务 | 高频用户 |

### 1.3 用户角色

| 用户角色 | 特征 | 核心需求 |
|----------|------|----------|
| **普通用户** | 偶尔租车 | 方便租车、价格优惠 |
| **PLUS会员** | 高频租车 | 专属折扣、优先服务 |
| **托管车主** | 托管车辆 | 稳定收益、车辆保值、自用便利 |

## 2. 页面结构与信息架构

### 2.1 底部导航设计

**底部标签栏 (Tab Bar)**:
- **首页**: 核心预订入口与内容推荐
- **社区**: UGC内容互动交流
- **托管中心**: 车辆托管运营服务
- **我的**: 个人中心与功能管理

**登录策略**：用户可自由浏览所有页面，业务操作时触发登录引导。

### 2.2 页面层级关系

#### 2.2.1 一级页面：首页

**页面定位**: 房车租赁业务的核心起点

**核心功能区域**:
1. **顶部公告栏**: 滚动通知，展示重要信息
2. **轮播广告位**: 活动推广和品牌展示
3. **房车预订模块（核心）**:
   - 取车城市选择
   - 取车门店选择
   - 异地还车选项
   - 取/还车日期时间选择
   - 查询可用房车按钮
4. **功能入口区**: 主要功能的快捷入口
   - **特惠租车**: 跳转到特惠租车套餐列表页
   - **特惠商城**: 跳转到优惠券商城页面（核心入口）
   - **托管中心**: 跳转到托管中心首页
   - **营地预订**: 跳转到营地列表页
   - **房车旅游**: 跳转到旅游路线列表页
   - **PLUS会员**: 跳转到会员购买/会员中心页
   - **我的优惠券**: 跳转到我的优惠券列表页
   - **邀请好友**: 跳转到邀请中心页面
5. **推荐内容区**: 社区内容推荐、托管车辆推荐

#### 2.2.2 一级页面：社区

**页面定位**: UGC内容互动与业务预订结合

**核心功能区**:
1. **业务推荐入口**:
   - 营地预订推荐
   - 房车旅游推荐
2. **内容互动区**:
   - 攻略、体验、活动内容
   - 用户互动（点赞、评论、收藏）

#### 2.2.3 一级页面：托管中心

**页面定位**: 车辆托管运营服务的核心平台

**核心功能区**:
1. **顶部轮播/公告区**: 托管活动和重要通知
2. **我的托管收益卡片**:
   - 未登录：大按钮"立即托管赚租金"
   - 已登录：累计收益、今日待结算、本月预估收益、提现按钮
3. **我的托管车辆**: 车辆卡片展示（最多3台，超出显示"查看全部"）
4. **收益明细入口**: 收益明细、托管协议与帮助
5. **主要功能区**: 2×2网格布局
   - 我要托管闲置房车（自有车免费入驻）
   - 购车托管（保底+高分成）

#### 2.2.4 一级页面：我的

**页面定位**: 用户个人中心和功能管理入口

**核心功能区**:
1. **用户信息区**: 头像、昵称、会员状态
2. **订单管理**: 待付款、待取车等订单入口
3. **资产管理**: 优惠券、钱包余额、托管资产
4. **服务入口**: PLUS会员、收藏、客服、设置等

### 2.3 信息流转路径

**用户决策路径**:
1. **浏览阶段**：首页 → 功能入口 → 列表浏览
2. **决策阶段**：详情查看 → 参数选择 → 价格确认
3. **操作阶段**：信息填写 → 支付完成 → 订单生成
4. **服务阶段**：订单管理 → 服务使用 → 评价反馈

## 3. 核心功能需求

### 3.1 首页模块功能

**核心功能需求**:
- 公告栏：重要信息滚动展示
- 房车预订：城市、门店、租期条件设置
- 功能入口：各业务模块快捷访问
- 内容推荐：社区内容和众筹项目推荐

**关键业务要求**:
- 查询条件必须完整才能启用查询按钮
- 首次使用自动定位用户当前城市
- 营销内容按优先级展示（活动 > 会员 > 优惠券）

### 3.2 房车租赁模块

**核心功能需求**:
- **房车列表**: 按门店分组展示可用车辆
- **房车详情**: 车辆信息、配置、评价展示
- **保险选择**: 基础险、标准险、全险三种方案
- **订单确认**: 完整预订信息确认和支���

**关键业务要求**:
- **车辆展示**: 按门店距离分组，同门店按价格排序
- **保险方案**:
  - 基础险：第三者责任险
  - 标准险：基础险+车辆损失险（80%赔付）
  - 全险：标准险+车辆损失险（100%赔付）+驾意险
- **价格构成**: 基础租金 + 保险费 + 附加服务费 + 异地还车费

### 3.3 特惠租车模块

**核心功能需求**:
- **套餐列表**: 特价套餐展示和筛选
- **套餐详情**: 取还车门店、价格、租期、时间段详情
- **预订流程**: 取车日期选择、保险服务选择、订单确认
- **自动计算**: 根据固定租期自动计算还车时间

**关键业务要求**:
- **全要素固定**: 取车门店、还车门店、价格、租期完全固定
- **取车时间选择**: 用户只能在预设时间段内选择取车日期
- **保险选择**: 必须选择基础险、标准险或全险之一
- **附加服务**: 可选择门店特色服务和全局附加服务
- **名额限制**: 每个套餐有限定名额，先订先得
- **独立资源**: 不占用普通租赁车辆资源

### 3.4 营地预订模块

**核心功能需求**:
- 营地列表：按距离、评分、价格筛选
- 营地详情：设施、营位信息展示
- 营位预订：选择营位类型和时间
- 订单管理：预订订单处理

**关键业务要求**:
- **营位类型**: 标准营位、豪华营位A、豪华营位B
- **预订时间**: 入住时间14:00后，退房时间12:00前
- **取消政策**:
  - 入住前3天以上：全额退款
  - 入住前1-3天：退款50%
  - 入住当天：不退款

### 3.5 房车旅游模块

**核心功能需求**:
- 路线列表：旅游路线展示和筛选
- 路线详情：行程安排、价格、批次信息
- 路线预订：选择批次和填写预订信息
- 成团管理：成团状态跟踪和通知

**关键业务要求**:
- **成团人数**: 最少3-5人，最多10-15人
- **批次状态**: 招募中、已成团、满员、已出发、已结束
- **价格规则**: 按人计价，儿童（12岁以下）半价
- **取消政策**: 出发前7天以上全额退款，3-7天退款70%

### 3.6 社区互动模块

**核心功能需求**:
- 内容浏览：攻略、体验、活动内容
- 内容发布：图文、视频内容创建和分享
- 互动功能：点赞、评论、收藏、分享
- 个人主页：用户动态和关注管理

**关键业务要求**:
- **内容审核**: 发布内容需经过审核，24小时内完成
- **用户权限**: 未登录可浏览，登录后可互动
- **内容分类**: 攻略、体验、活动、问答四大类别
- **关注上限**: 最多关注500人

### 3.7 托管中心模块

**核心功能需求**:
- **自有车托管**: 闲置房车免费入驻托管运营
- **购车托管**: 购车托管，保底收益+高分成
- **车主自用**: 托管车辆车主自用申请和管理
- **收益管理**: 实时收益查看、提现、对账单下载
- **车辆状态**: 托管车辆实时状态和运营数据查看

**关键业务要求**:
- **自有车托管流程**:
  - 上传行驶证 + 车辆实拍照片（至少12张）
  - 后台人工审核（最长24小时）
  - 通过后签署电子托管协议
  - 选择就近合作门店交车
  - 上架出租

- **购车托管流程**:
  - 热门车型展示（上汽大通V90、览众勇士、隆翠等）
  - 在线计算器（分期期数、月供演示）
  - 填写购车申请
  - 后台审核
  - 保底3500元/月 + 超额70%分成（管理端可自定义）

- **车主自用申请**:
  - 自动默认取车门店 = 该车当前运营门店
  - 选择还车门店（支持异地还车，系统自动计算异地还车费）
  - 可申请减免异地还车费（1次/月）
  - 费用明细：租金0元 + 门店服务费（200-500元/次）+ 保险费（已包含）+ 异地还车费（可减免）+ 附加服务
  - 提交申请 → 平台24小时内审核通过 → 生成自用订单

- **收益与提现**:
  - 实时查看昨日收益、历史明细
  - 月度对账单（可下载PDF）
  - 一键提现（满100元，T+1到账）

- **车辆状态实时查看**:
  - 当前位置（只显示地址文本）
  - 出租日历（已租日期灰色）
  - 维保记录、保险状态
  - 当前租客订单（匿名）

**关键业务规则**:
- **保险政策**: 平台统保太平洋营运车辆保险，车主零保险成本
- **租金分成**:
  - 自有车：车主70%（含淡季11.15-3.15额外补贴最高1000元/月，管理端可自定义）
  - 购车托管：保底3500元/月 + 超额租金70%（管理端可自定义）
- **托管期限**: 最短6个月，可提前30天申请终止托管取回车辆
- **展示优先**: 所有托管车辆在租车列表页显示"托管优选"角标，排序优先
- **订单可见**: 车主可在"我的-订单"里看到自己车辆被租的所有订单和租客评价
- **自用减免**: 每年可申请1次异地还车费减免

### 3.8 会员服务模块

**核心功能需求**:
- 会员购买：PLUS会员购买和支付
- 权益管理：会员专属折扣和特权服务
- 续费管理：手动续费和自动续费
- 会员中心：会员状态和权益查看

**关键业务要求**:
- **会员费用**: 年费99元
- **专属权益**:
  - 房车租赁95折优惠（特惠套餐除外）
  - 免费取消订单特权（取车前24小时内）
  - 专属客服优先响应
  - 生日当月8折优惠券
- **续费规则**: 到期前3天可续费，支持自动续费
- **退款规则**: 购买后7天内可申请退款

### 3.9 支付结算模块

**核心功能需求**:
- **支付方式管理**: 微信支付、支付宝、余额支付
- **组合支付**: 多种支付方式组合使用
- **支付安全**: 支付密码、指纹、面部识别
- **交易记录**: 支付历史和明细查询
- **发票管理**: 发票申请和下载

**关键业务要求**:
- **支付方式**:
  - 微信支付：支持信用卡、借记卡、零钱
  - 支付宝：支持余额、花呗、银行卡
  - 余额支付：账户余额直接支付
- **组合支付**:
  - 最多支持2种支付方式组合
  - 优先使用余额，不足部分使用其他方式
  - 优惠券抵扣后计算组合支付
- **支付限制**:
  - 单笔支付限额：50,000元
  - 日累计支付限额：200,000元
  - 密码错误超过3次锁定24小时

### 3.10 用户中心模块

**核心功能需求**:
- 个人资料：基本信息编辑和头像上传
- 订单管理：各类订单查看和管理
- 收藏管理：房车、营地、内容收藏
- 常用人管理：常用联系人和证件信息
- 优惠券管理：我的优惠券查看和使用
- 积分管理：积分余额、获取记录、兑换记录
- 邀请中心：邀请好友、查看邀请记录和奖励
- 客服帮助：客服与帮助中心入口
- 设置中心：账号、通知、隐私设置

**关键业务要求**:
- **资料要求**: 预订时需要驾驶证资料（后台审核）
- **常用人**: 信息预填提高预订效率
- **设置项**: 修改即时生效

**优惠券管理功能**:
- **我的优惠券入口**: 用户中心资产管理区显示优惠券数量，点击进入优惠券列表
- **优惠券列表页**:
  - **Tab分类**: 未使用、已使用、已过期
  - **未使用列表**:
    - 按过期时间排序（即将过期的排在前面）
    - 显示优惠券面额、名称、使用条件、有效期
    - 即将过期提醒（7天内过期高亮显示）
    - 操作按钮：立即使用、查看详情、分享
  - **已使用列表**:
    - 显示使用时间、订单号
    - 点击可查看使用订单详情
  - **已过期列表**:
    - 显示过期时间
    - 灰色显示，不可操作
- **优惠券详情页**:
  - 优惠券完整信息展示
  - 使用规则说明
  - 适用范围说明
  - 立即使用按钮（跳转到对应业务页面）
  - 分享按钮（分享得券功能）
- **过期提醒**:
  - 优惠券7天内过期时推送提醒
  - 优惠券过期当天再次提醒
  - 提醒消息显示在消息中心

**积分管理功能**:
- **积分余额卡片**: 用户中心资产管理区显示当前积分余额
- **积分明细页**:
  - **收入记录**: 托管收益、评价奖励、签到、活动等
  - **支出记录**: 兑换优惠券
  - **每条记录**: 显示时间、来源/用途、积分变动、余额

2. **浏览筛选车辆**（车辆列表页）: 按门店分组展示
3. **查看车辆详情**（车辆详情页）: 配置、价格、评价
4. **确认订单信息**（订单确认页）: 保险选择、附加服务
5. **完成支付订单**: 支付选择、订单生成
6. **取车使用**: 到店取车、开始租赁
7. **异常处理**: 续租申请、事故/道路救援统一申报
8. **还车完成**: 门店还车、结束租赁

**异常处理子流程**：
- **续租申请**：租期内用户从订单详情进入续租入口，选择额外天数→系统校验车辆是否可延长→展示新的还车时间与费用→发起续租支付或提交人工审核 → 更新订单时间轴。
- **事故/故障申报**：提供事故类型选择、图文描述、位置信息和联系方式；提交后生成记录并通知客服，订单状态保持但追加处理记录。
- **事故/道路救援申报**：提供图文描述、定位上传、照片取证及电话联络入口，提交后客服统一受理，包括道路救援、拖车、保险理赔等场景。

**关键流程要求**：
- **强制查询条件**: 城市、门店、租期三个条件必须完整
- **异地还车**: 支持部分城市，需额外费用
- **保险选择**: 必须选择基础险、标准险或全险之一
- **库存预留**: 选择车辆后预留30分钟
- **取消政策**:
  - 取车前24小时以上：全额退款
  - 取车前24小时内：扣除10%违约金
  - 取车后：按实际使用天数计算

### 6.2 自有车托管流程

**流程步骤**：
1. **了解托管**: 查看托管中心主页，了解托管政策和收益
2. **提交申请**: 填写车辆信息，上传行驶证和车辆照片（至少12张）

**关键流程要求**：

- **审核时效**: 1-3个工作日完成审核
- **收益分成**: 车主70%，平台30%

## 7. 交互体验需求

### 7.1 页面交互要求

**导航交互**：
- 所有次级页面左上角必须提供返回按钮
- 底部标签栏切换响应及时，状态明确
- 页面转场流畅，方向感明确

**列表交互**：
- 支持下拉刷新和上拉加载更多
- 筛选条件展开收起动画流畅
- 列表项点击反馈及时明显

**表单交互**：
- 输入框聚焦状态明显
- 验证错误提示清晰友好
- 必填项标识明确（红色星号）

### 7.2 数据展示要求

**信息层级**：
- 重要信息突出显示，次要信息弱化处理
- 价格信息醒目，包含单位说明
- 状态信息使用不同颜色区分

**加载状态**：
- 页面初次加载显示骨架屏
- 数据更新时显示加载指示器
- 网络异常时提供重试选项

**空状态处理**：
- 无数据时显示友好的空状态提示
- 提供相应的操作引导
- 保持页面布局美观

### 7.3 操作反馈要求

**成功反馈**：
- 操作成功时显示成功提示
- 关键操作（如支付）显示详细结果
- 状态变更及时反映在界面上

**错误处理**：
- 网络错误时提供明确错误信息
- 业务规则违反时给出具体说明
- 提供问题解决建议或操作指引

**确认机制**：

## 8. 业务规则要求

### 8.1 预订规则

**房车租赁**：
- 强制条件查询（城市、门店、时间）
- 必须选择保险
- 支持取消，按时间段扣除费用
- 取车前24小时以上全额退款
- 取车前24小时内扣除10%手续费
- 取车后按实际使用天数计算

**特惠租车**：
- 固定套餐，不支持个性化修改
- 独立车辆资源，不占用普通租赁
- 特殊取消政策，按时间梯度退款
- 出发前7天以上：全额退款
- 出发前3-7天：退款70%
- 出发前1-3天：退款30%
- 出发当天：不退款

**营地预订**：
- 按天预订，支持多天
- 营位库存实时管理
- 入住前3天以上：全额退款
- 入住前1-3天：退款50%
- 入住当天：不退款

**房车旅游**：
- 按批次组织，需要成团
- 成团前取消全额退款
- 不成团时全额退款或改期
- 出发前7天以上：全额退款
- 出发前3-7天：退款70%
- 出发前1-3天：退款30%

### 8.2 支付规则

**支付方式**：
- 微信支付、支付宝支付、余额支付
- 支持组合支付
- 15分钟内完成支付

**押金规则**：
- 车辆押金：到店支付，还车时退还
- 违章押金：30天后无违章自动退还

**退款规则**：
- 原路退回：支付渠道原路退回
- 退款时效：1-3个工作日到账
- 部分退款：按比例退回各支付渠道

### 8.3 托管规则

**自有车托管规则**：
- 车辆要求：车龄≤8年，无重大事故，手续齐全
- 照片要求：至少12张实拍照片（外观、内饰、细节）
- 审核时效：1-3个工作日完成审核
- 收益分成：车主70%，平台30%
- 淡季补贴：11.15-3.15期间最高1000元/月保底收益
- 托管期限：最短6个月，最长不限

**购车托管规则**：
- 申请人要求：年满22周岁，信用良好，有稳定收入
- 保底收益：3500元/月保底收益
- 超额分成：超过保底部分车主70%，平台30%
- 审核时效：3-5个工作日完成资质审核
- 分期支持：支持12/24/36期分期付款
- 托管期限：与分期期限一致

**车主自用规则**：
- 租金优惠：车主自用租金为0元
- 服务费：需支付门店服务费200-500元/次
- 保险费：已包含在托管协议中，无需额外支付
- 异地还车：支持异地还车，可享受1次/月减免
- 审核时效：1个工作日内完成审核
- 取消政策：取车前24小时可免费取消

**收益发放规则**：
- 发放周期：按月发放，次月5号到账
- 结算方式：自动结算到托管车主账户余额
- 提现规则：单次最低提现100元，T+1到账
- 税费处理：平台代扣代缴个人所得税

### 8.4 会员权益规则

**PLUS会员**：
- 会员费：年费99元
- 专属折扣：房车租赁95折
- 免费取消订单特权
- 专属客服支持

**权益叠加规则**：
- 会员折扣 vs 优惠券：会员折扣优先，优惠券在会员折扣后使用
- 多种优惠券：仅限使用1张优惠券，不可叠加使用（日租抵扣券可叠加使用）
- 积分抵扣：积分不可直接使用，只能在特惠商城兑换优惠券
- 特价商品：特惠套餐等特价商品不享受会员折扣

### 8.5 积分系统规则

**积分获得规则**：
- 托管收益：托管车主每月收益的1%转化为消费积分（仅首次托管，包括自有车和购车托管）
- 评价反馈：优质评价获得10-50积分，图片评价额外20积分（限定托管车主）
- 签到打卡：每日签到获得2积分，连续签到额外奖励（限定托管车主）
- 活动参与：参与平台活动获得5-100积分（限定托管车主）
- 推荐好友：成功推荐新用户注册获得50积分（限定托管车主）
- 租车消费：普通用户租车消费100元返1积分

**积分使用规则**：
- 积分兑换：1积分=1元，消费返积分100元返1积分，积分仅可兑换优惠券
- 抵扣限制：不可抵扣现金，仅用于特惠商城兑换优惠券
- 有效期：积分根据用户身份获取，不设定有效期
- 积分转让：不支持积分转让或赠送

### 8.6 内容审核规则

**审核范围**：
- 社区发布内容（攻略、体验、活动、问答）
- 用户评论和评价
- 用户头像和昵称
- 举报内容

**审核标准**：
- 内容相关性：必须与房车、旅游、露营相关主题
- 合法性：符合国家法律法规，禁止违法内容
- 质量要求：内容完整、有价值，避免无意义灌水
- 图片规范：图片清晰、真实，无水印、无侵权

**审核流程**：
- 自动审核：AI系统初步筛查违规内容
- 人工审核：24小时内完成人工审核
- 审核状态：待审核、审核通过、审核拒绝
- 申诉机制：拒绝后可申诉，48小时内重新审核

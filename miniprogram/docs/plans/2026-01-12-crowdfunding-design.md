# 众筹托管功能优化设计方案

> 创建日期：2026-01-12
> 状态：待实现

---

## 一、背景与目标

### 1.1 当前问题

1. **委托机制不明确**：众筹完成后如何引导用户托管给叨叨房车运营，存在法律风险
2. **自用规则缺失**：众筹车辆多车主场景下，如何保证合理自用需求与运营效率平衡

### 1.2 优化目标

- 明确三种委托方案，让用户自主选择
- 建立完善的车主自用规则体系
- 完善退出与解散机制

---

## 二、委托方案设计

### 2.1 三种委托方案

| 方案 | 车辆登记 | 叨叨出资 | 叨叨收益 | 风险说明 |
|------|----------|----------|----------|----------|
| **A. 叨叨参与众筹** | 叨叨名下 | 出资占股 | 分红 + 服务费 | 叨叨风险共担 |
| **B. 叨叨仅受托** | 叨叨名下 | 不出资（代持） | 仅服务费（含代持风险） | 叨叨承担代持风险 |
| **C. 用户自行委托** | 用户名下 | 不出资 | 仅服务费/不参与 | 用户自担风险 |

### 2.2 方案说明

- **方案 A**：叨叨作为众筹参与方出资占股，车辆登记在叨叨名下，按股份分红并收取运营服务费
- **方案 B**：叨叨不出资但代持车辆（登记在叨叨名下），仅收取运营服务费，代持风险已包含在服务费中
- **方案 C**：车辆登记在某众筹用户名下，叨叨仅提供运营服务或不参与，需向用户展示法律风险声明

## 三、众筹项目规则

### 3.1 基础规则

| 规则项 | 内容 |
|--------|------|
| **最低份额** | 单人最低持有 10% |
| **最多人数** | 10 人（由最低份额决定） |
| **发起人要求** | 至少认购 10%（最低份额） |
| **成立条件** | 100% 份额全部认购完成 |
| **资金托管** | 平台托管账户 |
| **失败处理** | 全额退还；发起人可申请延期 |

---

## 四、车主自用规则

### 4.1 自用时长

| 规则项 | 内容 |
|--------|------|
| **年度额度** | 60 天/12个月 |
| **分配方式** | 按众筹份额比例分配 |
| **超出处理** | 按正常租赁价格支付 |
| **预约方式** | 先到先得，随时可约 |
| **预约限制** | 需车辆空闲（不能踢掉已有租客） |

### 4.2 自用期间费用

| 车主自付 | 平台暂停支付 |
|----------|--------------|
| 保险 | 托管分润 |
| 门店服务费 | 保底费用 |
| 异地还车费 | 淡季补贴 |

### 4.3 自用期间风险承担

| 项目 | 承担方 |
|------|--------|
| 常规保养 | 车主 |
| 维修费用 | 车主 |
| 事故贬值 | 车主 |
| 超保险费用 | 车主 |
| 事故损失补偿 | 按份额赔付其他车主 |

---

## 五、托管期间责任

| 项目 | 规则 |
|------|------|
| **维修/保险** | 平台承担 |
| **停运期间** | 平台暂停支付托管费 |
| **恢复运营** | 恢复支付 |

## 六、退出与解散机制

| 项目 | 规则 |
|------|------|
| **份额转让** | 份额市场自由转让 |
| **解散发起** | 任何车主可发起 |
| **解散条件** | 2/3 份额同意 |
| **车辆处置** | 市场价出售 |
| **款项分配** | 按份额比例 |

## 八、页面改进设计

### 8.1 发起众筹页面 (`create.vue`)

**新增委托方案选择步骤：**

```
流程：选择车型 → 【选择委托方案】→ 设置众筹信息 → 确认发起
```

- 三卡片式选项 A/B/C
- 方案 C 选择后弹出风险声明弹窗
- 用户需确认知悉风险后才能继续

### 8.2 众筹协议页面

**动态协议内容：**

- 根据 A/B/C 方案生成对应条款
- 包含自用规则条款（60天额度、费用承担）
- 包含退出机制条款（2/3同意解散规则）

### 8.3 我的众筹页面 (`my-crowdfunding.vue`)

**新增功能：**

- 自用额度显示：已用 X 天 / 总额度 Y 天
- 自用预约入口
- 自用记录查看

### 8.4 自用预约页面（新增）

| 模块 | 说明 |
|------|------|
| 额度提示 | 剩余额度 X 天 |
| 日期选择 | 起止日期选择器 |
| 可用性检查 | 提交时校验车辆是否空闲 |
| 费用预估 | 保险 + 服务费 + 异地费 |
| 超额提示 | 显示需付租赁费 |

### 8.5 项目详情页 (`project-detail.vue`)

**新增功能：**

- 委托方案标识：显示 A/B/C 方案
- 运营数据面板：出租天数、收益、维修记录
- 车主列表：各车主份额占比

### 8.6 解散投票页面（新增）

| 模块 | 说明 |
|------|------|
| 发起说明 | 填写解散原因 |
| 投票状态 | 各车主投票情况 |
| 进度条 | 当前/需要的 2/3 份额 |
| 截止时间 | 投票有效期 |

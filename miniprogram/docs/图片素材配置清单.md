## 四、图片尺寸规范

### 4.1 通用尺寸标准

| 组件类型 | 推荐尺寸 | 圆角 | 其他样式 |
|---------|---------|------|---------|
| **App Logo** | 160×160rpx | 16rpx | 阴影效果 |
| **用户头像（大）** | 120×120rpx | 50%（圆形） | 白色边框4rpx |
| **用户头像（中）** | 72×72rpx | 50%（圆形） | 背景色#F0F0F0 |
| **用户头像（小）** | 32×32rpx | 50%（圆形） | 背景色#EEE |
| **轮播图Banner** | 750×480rpx | 0 | 指示器 |
| **车辆卡片图** | 750×500rpx | 12rpx | 背景色#F2F3F5 |
| **订单缩略图** | 160×120rpx | 12rpx | 背景色#F2F3F5 |
| **社区内容卡片** | 220×160rpx | 16rpx | 阴影效果 |
| **社区瀑布流** | 宽度自适应 | 16rpx | 高度自适应 |
| **营地/线路卡片** | 750×400rpx | 16rpx | 阴影效果 |
| **空状态图标** | 200×200rpx | 0 | 透明背景 |
| **第三方登录图标** | 88×88rpx | 50%（圆形） | 背景色#F5F5F5 |
| **功能图标** | 160×160rpx | 0 | 透明背景 |
| **会员背景** | 750×400rpx | 16rpx | 渐变效果 |

### 4.2 图片格式建议

| 图片类型 | 推荐格式 | 说明 |
|---------|---------|------|
| **Logo、图标** | PNG | 支持透明背景 |
| **照片、展示图** | JPG | 文件体积小 |
| **会员背景** | PNG | 支持渐变和透明 |
| **空状态插图** | PNG/SVG | 矢量图更清晰 |

### 4.3 图片质量要求

| 场景 | 分辨率 | 文件大小 |
|------|--------|---------|
| **轮播图** | 1500×960px（2倍图） | < 200KB |
| **车辆展示图** | 1500×1000px（2倍图） | < 300KB |
| **用户头像** | 240×240px（2倍图） | < 50KB |
| **图标** | 176×176px（2倍图） | < 30KB |
| **空状态图** | 400×400px（2倍图） | < 100KB |

## 七、图片优化建议

### 7.1 性能优化

1. **压缩图片**：使用 TinyPNG 或 ImageOptim 压缩
2. **懒加载**：使用 `lazy-load` 属性
3. **WebP 格式**：支持的平台使用 WebP
4. **CDN 加速**：生产环境使用 CDN

### 7.2 用户体验优化

1. **占位图**：加载前显示占位图
2. **渐进式加载**：先显示低质量图，再加载高清图
3. **错误处理**：图片加载失败显示默认图
4. **尺寸适配**：根据设备像素比提供不同尺寸

## 九、常见问题

### Q1: 图片显示不出来怎么办？

**A**: 检查以下几点：
1. 文件路径是否正确（注意大小写）
2. 文件是否存在于 `/static/` 目录
3. 图片格式是否支持（PNG、JPG、GIF）
4. 文件名是否包含特殊字符

### Q2: 图片显示模糊怎么办？

**A**:
1. 提供 2 倍图（实际尺寸 × 2）
2. 检查 `mode` 属性是否正确
3. 避免过度拉伸图片

### Q3: 如何批量替换图片？

**A**:
1. 保持文件名不变，直接替换文件
2. 使用全局搜索替换路径
3. 重新编译项目

### Q4: 图片加载太慢怎么办？

**A**:
1. 压缩图片文件大小
2. 使用 CDN 加速
3. 启用图片懒加载
4. 使用 WebP 格式

## 1. 产品概述

### 1.1 产品定位

叨叨房车小程序专注于房车租赁服务，为用户提供便捷的房车预订、营地预订、房车旅游、社区互动等一站式服务。

### 1.2 核心业务模块

| 业务模块 | 功能描述 | 目标用户 |
|----------|----------|----------|
| **房车租赁** | 多样化房车租赁服务 | 个人旅行者、家庭出游 |
| **特惠租车** | 固定路线特价服务 | 价格敏感用户 |
| **营地预订** | 房车旅行住宿服务 | 房车租赁用户 |
| **房车旅游** | 个性化路线服务 | 高端用户 |
| **社区互动** | UGC内容社区 | 房车爱好者 |
| **众筹房车** | 房车众筹和收益分润 | 众筹车主 |
| **用户中心** | 完整用户服务体验 | 所有用户 |

### 1.3 用户角色

| 用户角色 | 特征 | 核心需求 |
|----------|------|----------|
| **普通用户** | 偶尔租车 | 方便租车、价格优惠 |
| **PLUS会员** | 高频租车 | 专属折扣���优先服务 |
| **众筹车主** | 参与众筹 | 稳定回报、资金安全 |

### 1.4 全局导航

**底部导航**：
- 首页：房车预订、推荐内容（包括众筹项目）
- 社区：UGC内容互动交流
- 客服：帮助中心、在线客服（简化版，无搜索功能）
- 我的：个人中心、功能管理

**重要功能调整**：
- **移除首页搜索功能**：简化页面，突出核心预订功能
- **首页增加众筹项目推荐**：在推荐内容区增加众筹房车项目展示
- **移除客服搜索功能**：客服页面保留基础帮助功能，不包含问题搜索
- **移除车辆对比功能**：简化车辆列表操作，保留收藏和详情查看

## 2. 核心业务流程

### 2.1 房车租赁流程

**流程步骤**：
1. **设置条件**：用户在首页设置取车城市、取车门店、是否异地还车、取车时间、还车时间
2. **查询车辆**：点击"查询可用房车"获取符合条件的车辆
3. **浏览选择**：查看房车列表，选择合适车辆
4. **确认预订**：填写信息，选择保险和附加服务，确认异地还车费用（如有）
5. **支付订单**：完成支付，进入订单确认
6. **取车使用**：到店取车，开始租赁
7. **异常处理**：处理续租申请、事故报告、车辆故障等异常情况
8. **还车完成**：到指定门店还车，订单完成

**关键规则**：
- **强制查询条件**：城市、门店、取车时间三个条件必须全部设置才能查询车辆
- **异地还车规则**：
  - 支持异地还车的城市由后台配置，部分车型不支持异地还车
  - 异地还车需支付额外费用，费用在订单确认页显示
  - 特惠套餐可能包含免费异地还车服务
- **车辆分组展示**：查询结果按门店距离分组，同门店车辆按价格从低到高排序
- **价格计算规则**：基础租金 + 保险费用 + 附加服务费 + 异地还车费（如适用），不同时段价格不同
- **库存预留机制**：选择车辆后预留30分钟，超时自动释放
- **保险选择强制**：必须选择基础险、标准险或全险中的一种才能完成预订
- **订单取消规则**：
  - 取车前24小时以上：全额退款
  - 取车前24小时内：扣除租金10%作为违约金
  - 取车后：按实际使用天数计算费用，不足1天按1天计算

### 2.1.1 异常情况处理流程

**续租申请流程**：
- **触发场景**：用户在租期内需要延长租赁时间
- **申请规则**：必须在当前租期结束前24小时申请
- **处理流程**：
  1. 用户在订单详情页点击"申请续租"
  2. 选择续租天数，系统按当前价格标准计算续租费用
  3. 填写续租原因，提交申请
  4. 后台审核车辆后续预订情况
  5. 审核通过后，用户支付续租费用
  6. 订单租期自动延长，订单状态更新
- **限制条件**：如车辆后续已被预订，则无法续租

**事故处理流程**：
- **触发场景**：租赁期间发生交通事故或车辆损坏
- **处理流程**：
  1. 用户在订单页点击"事故报告"
  2. 填写事故信息（时间、地点、经过）
  3. 上传现场照片和相关证件
  4. 系统立即通知客服和门店
  5. 客服联系用户，指导现场处理
  6. 根据情况安排道路救援或维修
  7. 后台评估责任和费用
  8. 处理保险理赔，用户承担保险免赔额部分
- **重要规则**：事故发生后必须立即报告，严重事故需报警处理

**车辆故障处理流程**：
- **触发场景**：车辆发生非事故性故障（如机械故障、电器故障）
- **处理流程**：
  1. 用户在订单页点击"道路救援"
  2. 选择故障类型：无法启动、轮胎问题、电池问题、其他故障
  3. 系统自动定位用户位置
  4. 安排最近的合作维修点或道路救援服务
  5. 维修人员现场处理或拖车至维修点
  6. 根据故障情况提供备用车辆或延长租期
- **服务保障**：提供24小时道路救援服务，基础救援费用由平台承担

### 2.2 众筹购买流程

**流程步骤**：
1. **浏览项目**：查看众筹项目列表和详情
2. **选择份额**：确定购买份额数量
3. **确认信息**：填写购买人信息，选择抵扣券
4. **签署协议**：阅读并同意众筹协议
5. **完成支付**：支付购买款项
6. **获取份额**：获得众筹份额，开始收益

**关键规则**：
- **购买数量规则**：单次最低购买1份，单次购买上不封顶，众筹期内可多次购买
- **众筹成功判定**：达到目标金额或众筹期结束即视为成功，未达到目标金额全额退款
- **收益分配规则**：众筹成功次月开始计算收益，每月5号发放上月收益
- **抵扣券使用规则**：仅限持有"众筹车主"标签的用户使用，每笔交易限用1张
- **协议签署规则**：必须阅读并同意众筹协议才能完成支付
- **资金安全规则**：众筹资金由第三方支付平台托管，项目成功后划转至项目方

### 2.3 份额交易流程

**流程步骤**：
1. **浏览市场**：查看份额交易市场信息
2. **选择操作**：选择买入或卖出操作
3. **确认交易**：填写交易信息，确认手续费
4. **执行交易**：完成交易撮合和资金划转
5. **更新份额**：更新份额持有情况

**关键规则**：
- **价格形成机制**：买卖双方报价撮合成交，价格实时更新
- **交易手续费**：买卖双向收取0.5%手续费，最低1元起收
- **涨跌幅限制**：单日涨跌幅限制为±10%，超出范围暂停交易
- **风险控制机制**：
  - 价格异常波动时自动熔断暂停交易
  - 大额交易（单笔超过1000份）需要人工审核
  - 恶意操纵市场行为的监控和处罚
- **交易时间规则**：交易日9:30-15:00，非交易日暂停交易
- **资金到账规则**：卖出资金T+1日到账，买入即时确认份额

## 3. 功能模块需求

### 3.1 首页模块

**核心功能**：
- **顶部公告栏**：单行滚动公告，展示活动通知、系统公告等重要信息
- **轮播广告**：活动推广和品牌展示（移除搜索功能）
- **房车预订**：取车城市、取车门店、异地还车选项、取车时间、还车时间条件设置和查询
- **金刚区**：主要功能模块入口（6个功能入口）
- **推荐内容**：优质房车、营地、众筹项目推荐

**关键业务规则**：
- **公告栏规则**：
  - 单行文字横向滚动显示
  - 优先展示紧急公告和活动通知
  - 点击公告可查看详情
  - 支持多条公告轮播切换
- **查询条件验证**：取车城市、取车门店、取车时间三个条件必须全部填写完整才能启用查询按钮
- **异地还车选项**：
  - 勾选异地还车后显示还车城市和门店选择字段
  - 异地还车费用在查询结果页面和订单确认页显示
  - 部分车型不支持异地还车，系统会自动过滤
- **城市定位规则**：首次使用自动获取用户当前城市，支持手动切换城市
- **门店展示规则**：只显示选中城市的门店，按距离用户当前位置由近及远排序
- **推荐内容规则**：
  - 房车推荐：基于用户所在城市和浏览历史推荐
  - 营地推荐：显示距离用户100公里内的热门营地
  - **众筹项目推荐**：推荐优质的众筹房车项目
  - 路线推荐：推荐用户所在城市出发的热门房车路线
- **广告位轮播规则**：优先展示当前城市的活动，无活动时展示品牌广告
- **功能入口权限**：未登录用户可浏览所有入口，点击后触发登录引导

### 3.2 房车租赁模块

**核心功能**：
- **房车列表**：按门店分组展示可用车辆
- **房车详情**：车辆信息、配置、评价展示
- **保险选择**：基础险、标准险、全险选择
- **附加服务**：门店特色服务和全局附加服务
- **订单确认**：完整订单信息确认和支付

**关键业务规则**：
- **同城门店规则**：只展示用户选择城市的门店，按门店距离分组，同一门店内车辆按价格排序
- **车辆状态规则**：只显示可租赁状态的车辆，已租出、维护中车辆不展示
- **异地还车标识**：
  - 车辆列表明确标识是否支持异地还车
  - 异地还车费用在车辆详情页和订单确认页显示
  - 部分特惠套餐可能包含免费异地还车服务
- **合作车辆标识**：明确区分自营车辆和合作车辆，标注服务差异
- **保险方案规则**：
  - 基础险：包含第三者责任险，不包含车辆损失险
  - 标准险：包含基础险+车辆损失险（80%赔付）
  - 全险：包含标准险+车辆损失险（100%赔付）+驾意险
- **附加服务分类**：
  - 门店特色服务：特定门店提供的独特服务（如免费接送）
  - 全局附加服务：所有门店都提供的服务（如儿童座椅）
- **价格显示规则**：显示日租金，总价根据租赁天数和异地还车费自动计算
- **车辆操作规则**：
  - 提供【收藏】功能，用户可收藏喜欢的车辆
  - 提供【详情】功能，查看车辆详细信息
  - 移除【对比】功能，简化操作流程
- **车辆信息展示**：
  - 车辆列表显示基本信息（图片、名称、价格、评分）
  - 详情页提供完整车辆信息和实拍图片
  - 显示车辆实时可用状态和预约状态

### 3.3 特惠租车模块

**核心功能**：
- **套餐列表**：特价套餐展示和筛选
- **套餐详情**：路线、价格、包含服务详情
- **套餐预订**：特殊预订流程和规则

**关键业务规则**：
- **套餐固定性规则**：路线、行程、价格完全固定，不支持任何个性化修改
- **套餐独特性规则**：特惠套餐与普通租赁价格差异明显，体现性价比优势
- **取消退款特殊政策**：
  - 出发前7天以上：全额退款
  - 出发前3-7天：退款70%
  - 出发前1-3天：退款30%
  - 出发当天：不退款
- **套餐使用规则**：必须在指定时间、指定地点取车还车，不可更改
- **与普通租赁关系**：特惠套餐是独立业务线，不占用普通租赁车辆资源
- **套餐容量限制**：每个套餐有限定名额，先订先得

### 3.4 营地预订模块

**核心功能**：
- **营地列表**：按距离、评分、价格筛选
- **营地详情**：营地设施、营位信息展示
- **营位预订**：选择营位类型和时间
- **订单管理**：营地预订订单处理

**关键业务规则**：
- **营位类型分类**：
  - 标准营位：基础停车位，可接入水电
  - 豪华营位：独立停车位+独立水电+卫生间
  - 豪华营位A：标准营位+独立卫生间+淋浴
  - 豪华营位B：标准营位+独立卫生间+淋浴+Mini厨房
- **预订时间规则**：支持按天预订，入住时间14:00后，退房时间12:00前
- **营位库存管理**：实时显示可预订营位数量，预订成功后立即扣减库存
- **取消退款政策**：
  - 入住前3天以上：全额退款
  - 入住前1-3天：退款50%
  - 入住当天：不退款
- **与房车租赁关联**：预订营地时可选择是否同时预订房车，支持打包优惠
- **营地设施使用**：预订成功后可使用营地公共设施，部分设施需额外付费

### 3.5 房车旅游模块

**核心功能**：
- **路线列表**：旅游路线展示和筛选
- **路线详情**：行程安排、价格、批次信息
- **路线预订**：选择批次和填写预订信息
- **成团管理**：成团状态跟踪和通知

**关键业务规则**：
- **成团人数规则**：每条路线有最少成团人数（通常3-5人），最多人数限制（通常10-15人）
- **批次管理规则**：
  - 每条路线设置多个出发日期批次
  - 批次状态：招募中、已成团、满员、已出发、已结束
  - 批次取消：出发前7天人数不足时取消并全额退款
- **价格规则**：按人计价，儿童（12岁以下）通常享受半价优惠
- **成团前取消政策**：出发前7天以上全额退款，3-7天退款70%，3天内退款30%
- **不成团处理规则**：出发前7天通知用户，可选择全额退款或改期其他路线
- **房车使用关联**：可选择单独参加路线（自带房车）或房车+路线打包优惠

### 3.6 社区互动模块

**核心功能**：
- **内容浏览**：攻略、体验、活动内容
- **内容发布**：图文、视频内容创建和分享
- **互动功能**：点赞、评论、收藏、分享
- **个人主页**：用户动态和关注管理

**关键业务规则**：
- **内容审核规则**：
  - 发布内容需经过人工审核，24小时内完成审核
  - 审核标准：内容相关性、合法性、质量要求
  - 审核状态：待审核、审核通过、审核拒绝
- **用户权限规则**：未登录用户可浏览所有内容，登录后才能点赞、评论、收藏、分享
- **内容推荐规则**：基于用户所在城市、浏览历史和兴趣标签推荐相关内容
- **用户关系规则**：
  - 关注上限：最多关注500人
  - 互相关注：双方关注后可查看私密内容
  - 黑名单：拉黑后互相看不到对方内容
- **互动权限规则**：只有关注用户才能对私密内容进行互动
- **内容分类规则**：攻略、体验、活动、问答四大类别，发布时必须选择分类

### 3.7 众筹车主模块

**核心功能**：
- **项目浏览**：众筹项目列表和详情
- **份额购买**：选择份额和支付流程
- **收益管理**：收益查看和提取
- **积分账户**：积分查看和兑换

**关键业务规则**：
- **项目展示规则**：众筹中、已成功、已失败项目分类展示
- **收益计算规则**：
  - 收益按月计算，次月5号发放到账户余额
  - 收益=项目投资金额×年化收益率÷12个月
  - 年化收益率根据项目风险等级确定（5%-15%）
- **收益提取规则**：最低提取50元，提现需要审核，审核通过后3-5工作日到账
- **积分获得规则**：
  - 众筹车主用户每月按投资金额1%获得消费积分
  - 积分有效期1年，过期自动清零
- **积分兑换规则**：100积分=1元，可兑换优惠券、房车折扣等
- **项目管理规则**：定期发布项目进展报告，重大事项及时通知投资者

### 3.8 份额交易模块

**核心功能**：
- **交易市场**：份额买卖信息展示
- **挂牌出售**：设置出售价格和数量
- **购买份额**：选择购买份额和支付
- **交易记录**：历史交易查询

**关键业务规则**：
- **交易时间规则**：交易日周一至周五9:30-15:00，法定节假日休市
- **价格涨跌幅限制**：单日涨跌幅±10%，超出范围暂停交易
- **委托订单规则**：
  - 限价委托：指定价格，按价格优先时间优先成交
  - 市价委托：按当前最优价格立即成交
- **最小交易单位**：单笔最少买卖1份，必须是整数份
- **资金清算规则**：
  - 买入：资金冻结，成交后扣除份额和手续费
  - 卖出：份额冻结，成交后资金T+1日到账可用
- **风险控制规则**：
  - 持仓单一项目不得超过总资产的50%
  - 异常交易行为监控和限制
- **信息披露规则**：实时显示最新成交价、涨跌幅、成交量等信息

# 第一部分：通用设计原则

## 🎯 核心设计原则

### 1. 友好礼貌

- 减少无关设计元素对用户目标的干扰
- 礼貌地向用户展示服务
- 友好地引导用户进行操作

### 2. 重点突出

- 每个页面都应有明确的重点
- 避免页面上出现与用户决策和操作无关的干扰因素
- 用户进入新页面时能快速理解页面内容

### 3. 流程明确

- 避免在用户操作流程中出现打断
- 保持流程的连贯性
- 清晰的步骤指示

### 4. 清晰明确

- 告知用户身在何处、可以往何处去
- 确保用户在页面中不迷路
- 导航明确，来去自如

### 5. 减少等待，反馈及时

- 页面加载时予以及时反馈
- 操作结果明确告知
- 异常状态清晰提示

### 6. 便捷优雅

- 减少用户输入
- 避免误操作
- 利用接口提升性能

### 7. 统一稳定

- 不同页面间保持统一性和延续性
- 尽量使用一致的控件和交互方式

# 第二部分：微信小程序设计规范

**官方文档**：https://developers.weixin.qq.com/miniprogram/design/

## 📐 布局规范

### 1. 导航规范

#### 1.1 小程序菜单

- 微信在所有小程序页面右上角放置官方菜单
- 开发者不可自定义菜单内容
- 可选择深浅两种配色以适应页面设计风格
- 设计界面时需预留该区域空间

#### 1.2 页面内导航

- 所有次级页面左上角提供返回上一级页面操作
- iOS 用户可通过界面边缘向右滑动返回
- Android 用户可通过物理返回键返回
- 导航应尽量简单
- 自有导航样式应与官方菜单保持差异

#### 1.3 标签分页（Tab）导航

- 标签数量不得少于 2 个，最多不得超过 5 个
- 建议标签数量不超过 4 项
- 一个页面不应出现一组以上的标签分页栏
- 小程序首页可使用微信提供的原生底部标签分页样式

### 2. 加载与反馈

#### 2.1 启动页加载

- 展示品牌标志（Logo）
- 加载进度指示由微信统一提供，不可更改

#### 2.2 页面下拉刷新

- 微信提供标准的页面下拉刷新能力和样式
- 开发者无需自行开发

#### 2.3 页面内加载反馈

**推荐使用局部加载反馈**（更有针对性，页面跳动小）：

- 局部加载反馈：只在触发加载的页面局部进行反馈
- 谨慎使用模态加载（会覆盖整个页面）
- 若载入时间较长，应提供取消操作和进度条
- 载入过程中应保持动画效果
- 不要在同一个页面同时使用超过 1 个加载动画

#### 2.4 结果反馈

- **图标型弹出提示**：轻量级成功提示，1.5 秒后自动消失
- **文字型弹出提示**：轻量化文字解释，1.5 秒后自动消失
- **模态对话框**：需要用户明确知晓的操作结果
- **结果页**：操作结果已是流程终结的情况

### 3. 异常处理

#### 3.1 异常状态提示

- 予以用户必要的状态提示
- 告知解决方案，使其有路可退
- 杜绝用户莫名其妙又无处可去的情况

#### 3.2 表单出错

- 在表单顶部告知错误原因
- 标识出错误字段提示用户修改

### 4. 便捷优雅

#### 4.1 减少输入

- 利用接口（如摄像头识别、地理位置）改善输入体验
- 让用户做选择而不是键盘输入
- 提供搜索历史快捷选项

#### 4.2 避免误操作

- 充分考虑控件热区面积
- 最适宜点击物理尺寸：7mm-9mm
- 使用微信标准控件库确保可用性

## 🎨 视觉规范

### 1. 字体

- 常用字号：22, 17, 15, 14, 12 (pt)
- 与系统字体保持一致

### 2. 字体颜色

| 用途     | 颜色值             | 说明             |
| -------- | ------------------ | ---------------- |
| 主要内容 | rgba(0, 0, 0, 0.9) | 标题、重要文字   |
| 次要内容 | rgba(0, 0, 0, 0.6) | 副标题、说明文字 |
| 辅助内容 | rgba(0, 0, 0, 0.4) | 提示文字、占位符 |
| 禁用内容 | rgba(0, 0, 0, 0.3) | 禁用状态文字     |

### 3. 品牌色

- 微信绿：#07C160
- 可根据业务需求自定义品牌色

# 第六部分：叨叨房车设计规范

## 🎨 品牌色彩系统

### 1. 主色调

#### 1.1 橙色主题 (Primary Orange)

**色值**: `#FF9F29`
**RGB**: `255, 159, 41`
**设计理念**: 温暖、活力、阳光，象征房车旅行的自由与热情

**使用场景**:

- ✅ 主要操作按钮（登录、提交、确认等）
- ✅ 重要信息强调（价格、优惠标签）
- ✅ 核心功能入口（预订、支付按钮）
- ✅ 进度指示器、选中状态
- ✅ 悬浮按钮（FAB）、徽章数字

**透明度变体**:

```scss
$primary-color: #ff9f29; // 100% - 主要按钮、强调元素
$primary-color-light: rgba(255, 159, 41, 0.1); // 10% - 浅色背景
$primary-color-medium: rgba(255, 159, 41, 0.2); // 20% - 悬停背景
$primary-color-hover: rgba(255, 159, 41, 0.5); // 50% - 按钮悬停
```

**代码示例**:

```scss
// 主要按钮
.primary-button {
  background: #ff9f29;
  color: #ffffff;
  border-radius: 12rpx;
  height: 88rpx;
  font-size: 32rpx;
  font-weight: 500;

  &:active {
    background: rgba(255, 159, 41, 0.8);
  }
}

// 价格标签
.price-tag {
  color: #ff9f29;
  font-size: 40rpx;
  font-weight: 600;
}
```

**无障碍对比度**:

- 白色背景上: 对比度 3.2:1 ✅ （符合 WCAG AA 大文字标准）
- 黑色文字: 对比度 2.8:1 ⚠️ （建议用于大号文字或非文字元素）

### 4. 色彩搭配指南

#### 4.1 ✅ 推荐搭配

**橙色 + 白色**:

```scss
// 主要按钮
background: #ff9f29;
color: #ffffff;
```

**紫色渐变 + 白色文字**:

```scss
// 会员卡片
background: linear-gradient(135deg, #8860d0 0%, #a78bdb 100%);
color: #ffffff;
```

**橙色 + 浅灰背景**:

```scss
// 价格标签
background: rgba(255, 159, 41, 0.1);
color: #ff9f29;
```

**蓝色 + 浅蓝背景**:

```scss
// 信息提示
background: rgba(75, 145, 255, 0.1);
color: #4b91ff;
```

---

#### 4.2 ❌ 禁忌搭配

- ❌ **橙色 + 红色**: 视觉冲突，避免同时使用
- ❌ **紫色 + 橙色**: 除非有明确层级关系（如会员专属优惠）
- ❌ **低对比度组合**: 如浅灰文字 + 白色背景
- ❌ **过多颜色**: 单个页面不超过 3 种主题色

---

### 5. 色彩使用优先级

**优先级排序**:

1. **橙色 (#FF9F29)**: 最高优先级，用于最重要的操作
2. **紫色 (#8860D0)**: 会员专属功能
3. **蓝色 (#4B91FF)**: 信息提示、次要操作
4. **绿色 (#67C23A)**: 成功反馈
5. **红色 (#FF4D4F)**: 警告、错误

**单页面色彩使用建议**:

- 主色调: 1 种（橙色或紫色）
- 辅助色: 最多 2 种
- 功能色: 按需使用（成功/错误）

---

### 6. 无障碍设计标准

**WCAG 2.1 对比度要求**:

| 文字大小                         | AA 级 | AAA 级 |
| -------------------------------- | ----- | ------ |
| 正常文字 (< 24rpx)               | 4.5:1 | 7:1    |
| 大文字 (≥ 24rpx 加粗 或 ≥ 30rpx) | 3:1   | 4.5:1  |
| 非文字元素 (图标、按钮)          | 3:1   | -      |

**叨叨房车色彩对比度检查**:

| 颜色组合          | 对比度 | 符合标准       |
| ----------------- | ------ | -------------- |
| #FF9F29 + #FFFFFF | 2.8:1  | ✅ AA 大文字   |
| #8860D0 + #FFFFFF | 5.2:1  | ✅ AA 正常文字 |
| #4B91FF + #FFFFFF | 4.5:1  | ✅ AA 正常文字 |
| #67C23A + #FFFFFF | 5.5:1  | ✅ AA 正常文字 |
| #FF4D4F + #FFFFFF | 4.2:1  | ✅ AA 正常文字 |

**建议**:

- 橙色主要用于按钮、图标等非文字元素
- 如需用于文字，建议字号 ≥ 32rpx 或加粗显示
- 紫色、蓝色、绿色、红色可用于正常文字

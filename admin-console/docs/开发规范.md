## 🎯 核心规范概览

### 胶水编程优先

- 能复用不造轮子，能连接不重写
- 优先使用成熟的开源库和官方 SDK

### 质量红线

- ✅ 必须通过 Linter（零警告零错误）
- ✅ 必须通过所有测试
- ✅ 必须通过类型检查（TypeScript）

## 🔧 Admin Console 特定规范

### 技术栈

- Vue 3.4.0 + Composition API
- Element Plus UI 组件库
- TypeScript 5.1.6
- Vite 构建工具

### Vue 3 组件规范

- 单个组件不超过 200 行
- 使用 Composition API（`<script setup>`）
- Props 必须定义 TypeScript 类型

### 开发命令

```bash
cd admin-console
npm run dev          # 端口 5174
npm run lint         # ESLint + Prettier
npm run type-check   # TypeScript 类型检查
```text

# 叨叨房车项目 Claude Code 配置

## AI 工作流

这个目录包含了针对叨叨房车项目的专门化AI工作流和命令系统。

### 可用命令

- `/setup-auth` - 设置身份认证系统
- `/setup-vehicles` - 配置车辆管理模块
- `/setup-orders` - 配置订单管理系统
- `/api-design` - API设计助手
- `/db-schema` - 数据库架构查看器
- `/test-runner` - 测试运行器
- `/deploy-helper` - 部署助手

### 工作流

- `feature-development` - 完整功能开发工作流
- `bug-fix` - Bug修复工作流
- `api-development` - API开发工作流
- `database-migration` - 数据库迁移工作流

### 项目专用提示词

- `daodao-context` - 项目上下文和架构信息
- `code-review` - 代码审查标准
- `testing-guidelines` - 测试指南
